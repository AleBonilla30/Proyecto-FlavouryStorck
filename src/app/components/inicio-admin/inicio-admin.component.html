Administación de Usuarios

Desde este panel, podrás administrar y gestionar lo referente a los usuarios, agregar, modificar o cambiar rol, o eliminarlos.

Administración de Recetas

Desde este panel, podras gestionar lo referente a las recetas como añadir nuevas, modificar las existentes (alérgenos, ingredientes, cantidades...) o eliminar recetas.

Gestión de Stock

Desde este panel, podrás gestionar el stock disponible, asi cómo modificarlo, en caso de no coincidir con la realidad, o eliminar los productos que se han desechado.
